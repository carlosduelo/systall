Automated Installation
======================

# Install a new machine

```
curl -L https://github.com/carlosduelo/systall/archive/master.tar.gz | tar xfz -
systall-master/run <profile>
```

# Update packages
```
curl -L https://github.com/carlosduelo/systall/archive/master.tar.gz | tar xfz -
systall-master/update <profile>
```
